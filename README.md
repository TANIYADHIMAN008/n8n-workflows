# n8n Workflows Collection

This repository contains automation workflows built using n8n, including AI agents, messaging automation, and API integrations.

## 🚀 Workflows Included

### 📧 AI Email Automation
- Automatically reads emails and generates responses using AI

### 📲 WhatsApp Automation
- Sends and receives automated messages via WhatsApp API

### 🤖 AI Agent Workflow
- Uses AI models with memory and tools for intelligent responses

### 📊 Google Sheets Automation
- Reads and updates data using APIs and automation

### 📅 Google Calendar Automation
- Creates and manages calendar events automatically

### 🤖 Telegram AI Workflow
- Processes images and generates AI-based responses

## 🛠️ Tools & Technologies
- n8n
- OpenAI API
- Gmail API
- Google Sheets API
- WhatsApp API
- Telegram API

## 📌 About
These workflows demonstrate real-world automation use cases and integration with AI and external APIs.
